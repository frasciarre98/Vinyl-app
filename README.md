# Vinyl Catalog

## Come avviare il server locale

Poiché questa interfacca web comunica con un server locale sul tuo computer, è necessario che questo server sia attivo mentre usi l'applicazione.

1. **Apri il Terminale**
2. **Assicurati di essere nella cartella del progetto** (dove si trova questo file)
3. **Esegui il comando:**
   ```bash
   npm run dev
   ```
4. Verrai un messaggio che dice "Local: http://localhost:5173/". A quel punto il sito è raggiungibile.

Per spegnere il server, premi `Ctrl + C` nel terminale.
